# Forge

🌐 **[Accéder au site](https://imfoxys.github.io/forge/)**

> Hébergé via GitHub Pages.

Application de suivi de musculation sur-mesure : séances guidées, historique des performances, progression dans le temps. 100% côté client (localStorage), pensée pour un usage mobile en salle.

## Stack

React 19 + TypeScript + Vite, Recharts pour les graphiques de progression.

## Commandes

- `npm run dev` — serveur de développement avec HMR
- `npm run build` — vérifie les types puis build de production
- `npm run lint` — ESLint sur tout le repo
- `npm run preview` — sert le build de production en local
- `npm run deploy` — build puis publie `dist/` sur GitHub Pages
